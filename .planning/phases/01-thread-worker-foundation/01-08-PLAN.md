<!-- async_setup_entry / async_unload_entry (after plan 06): -->
async_setup_entry(hass, entry) -> bool
  - creates syncer (queue=None), worker, calls syncer.bind_queue(worker.queue)
  - creates ingester with worker.queue
  - worker.start() → ingester.async_start() → await syncer.async_start()
  - rollback: if any start raises, stops started components in reverse order
  - stores HaTimescaleDBData(worker, ingester, syncer) in entry.runtime_data
  - returns True

async_unload_entry(hass, entry) -> bool
  - data.ingester.stop()          # sync, not awaited
  - await data.syncer.async_stop()
  - await data.worker.async_stop()
  - returns True
